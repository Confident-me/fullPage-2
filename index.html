无论工作、学习还是生活，这里都充满了新想法、热情与活力。对互联网有热情，却找不到组织吗？加入启维吧，这会是一个崭新的开始。

  • 运营推广人员

我们需要这样的你：

一、工作技能：

1. 熟练掌握php
2. 熟练掌握sql、熟悉ajax、基础js 、基础html、基础css
3. 熟悉MVC模式的大型开发框架，熟悉Yii框架者优先
4. 熟悉移动端接口开发、后端服务实现
5. 有3~4年或以上开发经验 (须附上可展示的成熟作品)
6. 有良好的代码习惯，注释习惯

二、工作态度：

除了专业技能我们更需要你的工作态度，我们认为沉下心来做事是网站发展的持续动力，也是个人不断完善自我实现梦想的必然要求。

1. 务实、负责任、有耐力有耐心。
2. 有时间观念，能够按时完成工作, 不拖延。
3. 能够快速融入新的工作模式。
4. 对产品负责，在意用户体验，能够主动完善产品的缺陷和漏洞

如果你还有以下任一开挂技能，我们将优先考虑：
1. IOS或安卓移动端开发经验
2. 有B2C、C2C网络商城开发经验
3. 有在线音乐收听、电台的开发经验
4. 有大型社区开发经验
